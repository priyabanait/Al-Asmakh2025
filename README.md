# Al-Asmakh-Privilege

Al-Asmakh Privilege Program - Frontend Application

## Description

A Next.js-based frontend application for the Al-Asmakh Privilege Program, featuring user dashboard, partner dashboard, and privilege management.

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- React Icons

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Project Structure

- `/src/app` - Next.js app router pages
- `/src/components` - React components
- `/public` - Static assets
- `/src/styles` - Global styles

## Deployment

This project is configured for deployment on Vercel.
