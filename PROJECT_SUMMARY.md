# Al-Asmakh Frontend Project Summary

## Project Overview

This project is a responsive Next.js website built with Tailwind CSS based on the provided Figma design. The website showcases a modern digital services company with various sections including hero banner, services, testimonials, and contact information.

## Components Implemented

1. **Header/Navigation**
   - Responsive navigation with mobile menu
   - Logo and navigation links
   - Call-to-action button

2. **Hero Section**
   - Main headline and description
   - Call-to-action buttons
   - Statistics display
   - Hero image with decorative elements

3. **Features/Services Section**
   - Grid layout of service cards
   - Icons for each service
   - Service titles and descriptions

4. **Testimonials Section**
   - Client testimonials with quotes
   - Client images, names, and positions
   - Card layout with decorative elements

5. **Footer Section**
   - Company information
   - Quick links
   - Services links
   - Contact information
   - Newsletter subscription
   - Social media links
   - Copyright and policy links

## Technologies Used

- **Next.js**: For server-side rendering and routing
- **Tailwind CSS**: For responsive styling and design
- **React**: For component-based UI development
- **JavaScript**: For functionality and interactivity

## Responsive Design

The website is fully responsive and works on:
- Mobile devices
- Tablets
- Desktop computers

## How to Run

1. Navigate to the project directory
2. Run the start script:
   ```
   ./start.bat
   ```
   Or manually:
   ```
   npm install
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser
